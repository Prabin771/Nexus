# Alliazo Pro — AI Developer Studio

A powerful, single-file AI coding workspace using **100% free services** — no API keys required.

## ✨ Free Services Used

| Service | What it powers | Cost |
|---------|---------------|------|
| **Puter.js** | GPT-4o, Claude, Gemini — free AI chat | 🆓 Free |
| **Puter.fs** | Cloud file storage & sync | 🆓 Free |
| **Pollinations.ai** | Backup AI chat (OpenAI/Mistral) | 🆓 Free |
| **Pollinations Images** | AI image generation | 🆓 Free |
| **Pyodide** | Python execution in browser | 🆓 Free |
| **Prism.js** | Syntax highlighting | 🆓 Free |
| **marked.js** | Markdown rendering | 🆓 Free |
| **JSZip** | ZIP file export | 🆓 Free |
| **Web Speech API** | Voice input | 🆓 Free (browser built-in) |

## 🚀 Features

- **Multi-model AI**: GPT-4o, Claude Sonnet, Gemini Flash via Puter.js (free)
- **Streaming responses**: Real-time token streaming
- **6 AI modes**: Code, Chat, Study, Debug, SQL, Creative
- **Live code sandbox**: Run HTML, JS, CSS, and Python in-browser
- **Python runner**: Full Python execution via Pyodide (no server needed)
- **AI image generation**: Puter AI + Pollinations fallback
- **Voice input**: Speak your prompts
- **Cloud file storage**: Save/load files via Puter.js cloud
- **Session management**: Local session history with titles
- **ZIP export**: Download all generated code files as a ZIP
- **Portfolio export**: Export chat as Markdown
- **Image upload**: Vision support (analyze images with AI)
- **Console terminal**: Live JS/Python error output
- **Dark/light theme**: Persists in localStorage

## 📦 How to Use

1. Open `index.html` in any modern browser
2. No installation, no API keys, no server needed
3. Sign into Puter.js (optional) for cloud file sync
4. Start building!

## 🔑 Puter.js Sign-in

Puter.js works without sign-in for AI features. Signing in (free at puter.com) enables:
- Cloud file storage
- Larger context limits
- Persistent sessions across devices
